# MacScreen

MacScreen 是一个 macOS 动态壁纸应用，可以把本地视频作为桌面动态壁纸播放。它当前面向 Apple silicon Mac，主打本地资源、低打扰、低负载的桌面美化体验。

项目现在仍处于 MVP 阶段，重点是把核心闭环做稳定：选择视频、预览素材、应用到桌面、停止播放、管理本地素材，并尽量不影响用户正常使用电脑。

## 项目特点

- 原生 macOS 应用，使用 SwiftUI + AppKit 实现
- 支持内置动态壁纸素材
- 支持导入用户自己的本地视频
- 支持静态缩略图预览，避免打开 App 就解码播放 4K 视频
- 支持一键应用视频到桌面背景层
- 支持所有屏幕同步播放动态壁纸
- 支持停止、暂停、继续和切换下一个动态壁纸
- 支持菜单栏常驻入口
- 支持可选的启动时恢复上次动态壁纸
- 支持可选的开机启动
- 支持可选的低电量暂停提醒
- 支持偏好设置页集中管理常用选项
- 支持内置浏览器下载资源，下载完成后自动尝试导入素材
- 支持素材搜索、分类筛选和收藏
- 支持删除自定义素材或隐藏内置素材
- 支持恢复被隐藏的内置素材
- 支持 Sparkle 自动更新
- 支持打包为 DMG 分发
- 支持素材卡片 3D hover 动效，可手动开关
- 支持右键菜单：应用、暂停、停止、查看位置、删除
- 支持重复文件导入拦截提示
- 支持全局音量控制：主窗口、桌面叠加层、菜单栏三处可调节

## 当前功能

### 动态壁纸资源

MacScreen 会读取内置视频素材，并在左侧列表中展示资源名称、分辨率、时长和缩略图。选中素材后，右侧会显示静态预览。

### 应用到桌面

用户选中一个视频后，可以点击“应用到桌面”，App 会把视频静音循环播放到桌面背景层，不会抢占当前正在使用的窗口。当前会在所有屏幕同步播放同一个动态壁纸。

### 菜单栏控制

MacScreen 提供菜单栏常驻入口，可以快速显示主窗口、添加视频、应用当前选择、暂停或继续播放、切换下一个动态壁纸、停止播放、调节音量、初始化资源和打开壁纸网站。菜单栏会显示当前播放或暂停的壁纸名称。

### 素材浏览

主界面以固定尺寸素材墙展示壁纸。鼠标悬停在素材上时会显示标题、自动标签、分辨率、时长、大小、收藏和应用按钮。用户可以搜索标题或标签，也可以通过顶部分类筛选素材。

### 启动和电源策略

用户可以在菜单栏中开启“启动时恢复上次壁纸”和“开机启动”。这两个选项默认关闭，避免 App 第一次启动就自动播放视频。

用户也可以开启“低电量时暂停并提醒”。当 Mac 使用电池供电且电量低于阈值时，App 会先暂停动态壁纸并弹窗，让用户选择停止壁纸、退出应用或继续播放。

这些选项也可以在 macOS 原生偏好设置页中集中管理。

### 本地视频导入

用户可以在 App 内添加自己的本地视频。导入的视频会复制到当前用户的 Application Support 目录，不会写回项目仓库。

当前支持的视频格式：

- mp4
- mov
- m4v

通过内置浏览器下载资源时，App 不限制下载来源；下载完成后会自动尝试导入支持的视频或 zip 资源包。导入成功后默认清理下载源文件，避免下载目录长期堆积。

### 素材删除和恢复

用户可以删除自己导入的视频，也可以把内置素材从当前列表中隐藏。如果误删了内置素材，可以通过“初始化资源”恢复。

### 自动更新

项目已经接入 Sparkle 2。配置好 Sparkle 密钥和 GitHub Pages 后，用户可以在 App 菜单中检查更新，并在本地完成新版下载和安装。

详细配置步骤见：

```text
AUTO_UPDATE_GUIDE.md
```

## 安装和使用

开发阶段可以从源码运行：

```bash
make run
```

也可以构建 DMG：

```bash
make package
```

打包产物位于：

```text
dist/MacScreen.dmg
```

如果从 GitHub Release 下载后被 macOS 拦截，可以右键 App 选择“打开”。正式分发时建议配置 Developer ID 签名和 Apple notarization。

## 设计原则

MacScreen 的核心目标不是“尽可能播放更多视频”，而是让动态壁纸在 macOS 上稳定、安静、可控地运行。

后续功能会优先遵循这些原则：

- 不在启动时自动播放大视频
- 不抢焦点
- 不遮挡用户正在使用的窗口
- 不强依赖 Desktop、Documents 等隐私目录权限
- 用户可以随时停止动态壁纸
- 默认体验要轻量、稳定、可恢复
